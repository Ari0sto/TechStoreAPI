# TechStore API

Backend API для онлайн-магазина техники и электроники. Проект представляет собой REST API сервис, обеспечивающий полный цикл управления электронной коммерцией: от администрирования товаров до оформления заказов клиентами.

## Функциональность

### Основные возможности
* **Каталог товаров:** Управление электроникой и периферией (ноутбуки, жесткие диски и т.д.).
* **Свойства товара:** Название, цена, описание, количество на складе.
* **Корзина и Заказы:** Оформление заказов авторизованными пользователями.
* **Soft Delete:** Реализовано "мягкое удаление" товаров (скрытие без фактического удаления из БД).

### Безопасность и Аутентификация
* **JWT (JSON Web Tokens):** Аутентификация пользователей.
* **Хеширование паролей:** Безопасное хранение данных.
* **Ролевая модель:**
    * **Admin:** Управление товарами (CRUD), просмотр всех пользователей, управление статусами заказов.
    * **Client:** Просмотр каталога, создание заказов.

## Технологический стек
* **Платформа:** .NET Core / ASP.NET Core
* **База данных:** MSSQL (LocalDB)
* **ORM:** Entity Framework Core (использование миграций)
* **Архитектура:** REST API

## API Документация (Postman)
В проекте предусмотрена коллекция запросов для Postman, которая поможет быстро протестировать API.
Файл коллекции находится в папке: `/Postman/TechStore.postman.collection.json`.

## Установка и запуск
1.  Клонируйте репозиторий.
2.  Настройте строку подключения в `appsettings.json`.
3.  Примените миграции для создания базы данных:
    ```bash
    dotnet ef database update
    ```
4.  Запустите проект.
